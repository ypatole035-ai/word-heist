# 🔓 WORD HEIST

> *The vault is locked. Six words stand between you and the exit. You have 90 seconds.*

WORD HEIST is a browser-based word puzzle game with a heist aesthetic. Crack six word locks under a countdown timer. Wrong answers trigger the security system. Too many mistakes and the whole operation goes into lockdown.

It's a word game — but it doesn't feel like one.

## How to Play

- Six word puzzles stand between you and the vault
- Solve all six before the **90 second timer** runs out
- Wrong answers cost you **8 seconds** and raise the **Security Level**
- Security Level fills up → **Lockdown** → Game Over
- You get **2 hints** per run — use them wisely
- Clear all six → vault cracked → mission complete

The pressure isn't just the timer. It's the security bar creeping up every time you guess wrong.

---

## Modes

- **Classic** — standard run, your score saved to leaderboard
- **Daily** — one puzzle set per day, same for everyone
- **Practice** — no timer, no pressure, just the words

---

## Features

- **3 Difficulty levels** — Easy, Normal, Hard
- **Security system** — wrong answers don't just cost time, they escalate danger
- **Vault Shop** — spend credits on themes that change the entire visual identity of the game
- **Achievements** — unlock badges across runs
- **High Scores** — local leaderboard per difficulty
- **Glitch aesthetic** — VT323 monospace font, neon colors, terminal-style UI
